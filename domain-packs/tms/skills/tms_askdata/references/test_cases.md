# 推荐问句参考

## 支持的首批问题

1. 本月发货申请了多少单？
2. 本月制单了多少单？
3. 当前有多少单在运输途中？
4. 本月已签收多少单？
5. 本月哪些承运商承运量最高？
6. 本月准时到货率是多少？
7. 最近 7 天哪些项目延期最多？
8. 本月不同运输方式的申请量分布如何？
9. 本月询价单有多少？
10. 本月哪些供应商报价次数最多？

## 复杂问题拆解

### 问题

本月哪些承运商准时率最高，发货量是多少？

### 拆解

- 时间范围：本月
- 主体维度：承运商
- 指标 1：准时率
- 指标 2：承运量
- 主事实表：`HR_DELIVER_FORM + HR_DELIVER_CARRY`

### 问题

最近 7 天运输中的单子里，哪些项目延期最多？

### 拆解

- 时间范围：最近 7 天
- 状态过滤：运输中
- 维度：项目
- 指标：延期单量

## 禁止回答的方向

- 删除、修改、插入数据
- 查询历史备份表
- 使用未定义关系拼接表
